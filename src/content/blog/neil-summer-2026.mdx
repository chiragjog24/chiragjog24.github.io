## Residential programs

### 1. JHU CTY — Foundations of Programming ⭐ *Top pick if budget allows*

| | |
|---|---|
| **Provider** | Johns Hopkins University, Baltimore, USA |
| **Track** | 🇺🇸 US · Selective |
| **Dates** | Session 1: Jun 28 – Jul 17 (3 weeks) |
| **Cost** | ~$7,600 tuition (~₹6.4L) + family travel ≈ **₹12–13L total** |
| **Content** | Python programming, algorithms, data structures, functions, classes, sorting. Project-based. |
| **Eligible** | Grades 5–12 · Requires SCAT qualifying test (90th+ percentile) |
| **Deadline** | Enroll by May 11 (Session 1) · Take SCAT first |
| **Credential** | CTY course completion certificate (widely recognized by US admissions) |

**Pros**
- Selective — SCAT test requirement is itself a credential
- Official JHU program with JHU faculty
- Strongest US admissions signal among all options
- Gateway to future CTY advanced courses (DSA)
- 3-week immersion with gifted peers from 60+ countries
- US campus visit opportunity (MIT, Harvard, CMU nearby)

**Cons**
- Most expensive option (₹12–13L with family travel)
- 3 weeks = more school missed
- Programming content can be learned for free online
- Less relevant for UK/Singapore admissions

### 10. USACO — USA Computing Olympiad ⭐ *Highest ROI activity*

| | |
|---|---|
| **Provider** | Online, worldwide participation |
| **Track** | Free · Competitive |
| **Dates** | Contests: Dec, Jan, Feb, Mar (year-round practice) |
| **Cost** | **Free** |
| **Content** | Competitive programming: Bronze → Silver → Gold → Platinum. Algorithms, data structures, problem solving under time pressure. |
| **Eligible** | Any pre-college student worldwide |
| **Deadline** | None — register anytime at usaco.org |
| **Credential** | Division placement (Gold/Platinum = top ~5% globally) |

**Pros**
- Completely free
- USACO Gold carries MORE admissions weight than any summer program
- Directly builds real coding skills
- Can be done from home year-round
- Resources: usaco.guide (free structured curriculum)

**Cons**
- No residential/social experience
- Requires sustained self-discipline over months
- Competitive — reaching Gold takes 1–2 years of practice
- Not a UK/Singapore admissions signal (US-focused)

- Math-focused, not CS/engineering specific
- Only one shot per year
- Qualifying for AIME is very difficult (~top 5%)
