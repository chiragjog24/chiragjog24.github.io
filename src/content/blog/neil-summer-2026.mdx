## Summer programs — detailed comparison

### 1. JHU CTY — Foundations of Programming ⭐ *Top pick if budget allows*

| | |
|---|---|
| **Provider** | Johns Hopkins University, Baltimore, USA |
| **Track** | 🇺🇸 US · Selective |
| **Dates** | Session 1: Jun 28 – Jul 17 (3 weeks) |
| **Cost** | ~$7,600 tuition (~₹6.4L) + family travel ≈ **₹12–13L total** |
| **Content** | Python programming, algorithms, data structures, functions, classes, sorting. Project-based. |
| **Eligible** | Grades 5–12 · Requires SCAT qualifying test (90th+ percentile) |
| **Deadline** | Enroll by May 11 (Session 1) · Take SCAT first |
| **Credential** | CTY course completion certificate (widely recognized by US admissions) |

**Pros**
- Selective — SCAT test requirement is itself a credential
- Official JHU program with JHU faculty
- Strongest US admissions signal among all residential options
- Gateway to future CTY advanced courses (Data Structures & Algorithms)
- 3-week immersion with gifted peers from 60+ countries
- US campus visit opportunity (MIT, Harvard, CMU all accessible from Baltimore)

**Cons**
- Most expensive option (₹12–13L with family travel)
- 3 weeks = more school missed
- Programming content can be learned for free online (CS50, MIT OCW)
- Less relevant for UK/Singapore admissions specifically

**SCAT registration steps:**
1. Create CTY account at cty.jhu.edu, add Neil as student
2. Receive eligibility number within 1 business day
3. Take Online SCAT from home (under 1 hour, no download needed)
4. Scores arrive in ~2 weeks (Prometric center: 3 days)
5. Enroll by May 11 for Session 1

## India-based programs — surprisingly strong options

### 4. Indian Computing Olympiad (ICO) ⭐ *Highest impact free activity*

| | |
|---|---|
| **Provider** | IARCS (Indian Association for Research in Computing Science) |
| **Track** | 🇮🇳 India · Competitive · **Free pathway to IOI** |
| **Dates** | ZIO: November 2026 · ZCO: November 2026 · INOI: February 2027 |
| **Cost** | ₹472 registration |
| **Content** | ZIO: Paper-based algorithmic thinking (no coding). ZCO: Programming contest. INOI: Advanced algorithms in C++. |
| **Eligible** | Any Indian school student, all boards including IGCSE |
| **Deadline** | Registration opens October 2026 |
| **Credential** | INOI medal → IOITC selection → IOI team (4 represent India) |

**Why this matters more than any summer program:**
- IOI is the CS equivalent of the Olympics — an IOI selection is a top-tier credential for MIT, Stanford, Cambridge, NUS
- INOI gold medallists get special consideration at NUS, NTU, CMI, IIIT Delhi/Hyderabad
- Completely free
- Can be done from Nashik — no travel required for ZIO/ZCO

**Preparation path:**
- April–October 2026: Learn C++, practice on CodeChef and Codeforces, study IARCS online material
- Join Indian Competitive Programming Summer Camp (~$20 for 8 weeks, mentored by Grandmasters)
- November 2026: Write ZIO + ZCO
- February 2027: Attempt INOI (if qualified)

## What admissions officers actually value — a reality check

For US universities (MIT, Stanford, CMU): **Selective competitions > Research > Selective programs > Pay-to-attend programs.** USACO Gold + IRIS/ISEF + a shipped personal project outweighs any combination of summer certificates.

For UK universities (Imperial, Cambridge): **GCSE/A-Level grades > Personal statement > Interview performance > UCAS points.** Summer programs provide personal statement material but won't compensate for weak academics.

For Singapore (NUS, NTU): **Academic grades > Competition results > Attending their own summer programs.** NUS/NTU familiarity from their summer school is a genuine differentiator.

For all: **A GitHub portfolio of original projects, competitive programming achievements, and a research paper carry more weight than any number of summer camp certificates.**
